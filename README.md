# 下载使用  
首先复制仓库链接, 下载仓库到本地  

```
https://github.com/taochunhui824/vue3-ts-demo
```  
cmd命令(每个人知识储备程度不同,我就这样尽可能详细讲吧, 希望能帮到你) 进入到vue3-ts-demo文件夹

执行下面命令, 下载项目所需要的依赖  

```
npm install
```
然后执行下面命令来启动项目  

```
npm run serve
```

每个人网速或者npm的镜像不同, 可能需要等待一会儿, 如果没有什么问题, 会显示下面内容

```
http://localhost:8080/
```
